# Screenshots Directory

## Required Screenshots

After completing Telegram live test, this directory should contain:

### 1. telegram_stop.png
- User message: "delete server files"
- Bot response: "⚠️ Execution STOPPED - Dangerous action detected"
- Shows STOP decision in action

### 2. telegram_hold_payment.png
- User message: "buy this product"
- Bot response: "⏸️ Execution HELD - Financial impact detected"
- Shows payment held before execution

### 3. telegram_allow.png
- User message: "show me files"
- Bot response: "✅ Execution ALLOWED - Safe action completed"
- Shows safe operation allowed

## Screenshot Requirements

Each screenshot should clearly show:
- User's input message
- Bot's response (decision + reason)
- Timestamp (optional but recommended)
- Bot username (for authenticity)

## Naming Convention

Use exact filenames as listed above to match documentation references.

## After Collection

Once all 3 screenshots are in place:
- Verify they match expected responses
- Check they're readable and clear
- Confirm no sensitive information visible
